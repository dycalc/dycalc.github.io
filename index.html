kali安装软件

最近ubuntu用的有点不顺，在编辑器里面老是切不出中文输入法。正好想起了之前用过的kali系统，感觉还不错。就直接换掉了。这次装系统在新建用户那块没有用默认的帐号而是新建了一个帐号，因为以前写rails的时候遇到过root帐号无法安装某个依赖的情况，kali系统的普通帐号在默认情况下无法获得管理员权限，这次我直接新建一个帐号，这样的做法类似于ubuntu，我感觉可能就不会遇到这样的坑了。
安装chromium

$ sudo apt-get update
$ sudo apt-get install chromium chromium-l10n

安装nodejs
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.29.0/install.sh | bash
$ nvm ls-remote
$ nvm install v5.9.0
$ nvm alias default 5.9.0
$ npm install -g cnpm --registry=https://registry.npm.taobao.org

安装谷歌拼音
$ sudo apt-get install fcitx-googlepinyin
在输入法中切换fcitx输入法

安装网易云音乐
$ sudo pip2 install NetEase-MusicBox
$ sudo apt-get install mpg123

下载atom
$ dpkg -i file.deb

之后在装hexo的时候也遇到了些麻烦，我执行了cnpm i或者npm i之后再执行hexo的命令会提示在我的项目中没有hexo,全局安装hexo也没有效果，在我把项目中的node_modules删除后重新执行cnpm i后可以使用hexo了。

之后又遇到了新的坑，生成的index没有内容，查看报错信息后发现是主题没有了，那是我clone的主题，github可能没有发上去。

雨巷

###戴望舒
撑着油纸伞，独自
彷徨在悠长，悠长
又寂寥的雨巷，
我希望逢着
一个丁香一样的
结着愁怨的姑娘。
她是有
丁香一样的颜色，
丁香一样的芬芳，
丁香一样的忧愁，
在雨中哀怨，
哀怨又彷徨；
她彷徨在这寂寥的雨巷，
撑着油纸伞
像我一样，
像我一样地
默默彳亍①着，
冷漠，凄清，又惆怅②。
她静默地走近
走近，又投出
太息③一般的眼光，
她飘过
像梦一般的，
像梦一般的凄婉迷茫。
像梦中飘过
一枝丁香的，
我身旁飘过这女郎；
她静默地远了，远了，
到了颓圮④的篱墙，
走尽这雨巷。
在雨的哀曲里，
消了她的颜色，
散了她的芬芳
消散了，甚至她的
太息般的眼光，
丁香般的惆怅④。
撑着油纸伞，独自
彷徨在悠长，悠长
又寂寥的雨巷，
我希望飘过
一个丁香一样的
结着愁怨的姑娘。

注释
①彳亍（chìchù）：小步慢走的样子。
②惆怅（chóuchànɡ）：伤感，失意。
③太息：出声叹息。
④颓圮（tuípǐ）：倒塌。
